# Python Learning

Basic Python scripts for learning fundamentals.

## Files
- `hello_world.py`: Classic "Hello, World!" print.
- `addition.py`: Addition function `add(a, b)` with demo/interactive.
- `hello_ways.py`: **15+ ways to print "Hello"** (basic to advanced).

## Run
```bash
python hello_world.py
python addition.py
python hello_ways.py  # See all print ways!
```

## Setup
- Python 3.x
- GitHub Desktop (installed) for repo "Python Learning".
